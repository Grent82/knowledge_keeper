# Knowledge Keeper

Private Wissensplattform fuer Audio-, Video- und spaeter Markdown-basiertes Wissensmanagement.

## Ziel

Knowledge Keeper ist ein privater Wissensspeicher mit Weboberflaeche. Der Schwerpunkt liegt auf:

- geschuetztem Zugriff per Login
- Verwaltung lokaler und externer Medienquellen
- Audio- und Videowiedergabe mit nutzerindividuellem Fortschritt
- selektiver Freigabe von Inhalten fuer weitere Nutzer
- spaeterer KI-Integration fuer Transkripte, Zusammenfassungen und Wissensnotizen

## Produktphasen

1. Medienplattform
2. KI-Transkription und Zusammenfassung
3. Markdown-Wissensbasis und Verlinkung
4. Semantische Suche, Wissensgraph und Cloud-Storage
