# Backend

Geplantes Backend fuer:

- Authentifizierung und Rollen
- Medienverwaltung
- Fortschritt und Status pro Nutzer
- Sichtbarkeitsregeln
- spaetere Transkriptions- und KI-Jobs

## Geplanter Stack

- Django
- Django REST Framework
- PostgreSQL
- Celery
