# arc42 05 Building Block View

## Level 1

### Frontend

- Medienbibliothek
- Such- und Filteroberflaeche
- Audio-/Video-Player
- Wissensnotizen und spaeter Graphsicht

### Backend API

- Authentifizierung
- Medienverwaltung
- Kategorisierung und Tags
- Fortschritt und Status
- Rechtepruefung
- Wissensnotizen

### Background Jobs

- Transkriptionsjobs
- Zusammenfassungsjobs
- Taggingjobs
- spaeter Importjobs fuer externe Quellen

- relationale Daten fuer Metadaten und Rechte
- Dateispeicher fuer Medien
- Markdown-Dateien fuer Wissensnotizen
- spaeter Vektorindex fuer semantische Suche
