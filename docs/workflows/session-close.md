# Session Close Workflow

## Pflichtschritte

1. relevante Checks ausfuehren
2. Doku- oder ADR-Updates fertigstellen
3. Folgearbeit als Bead anlegen
4. Bead schliessen oder sauber im Status lassen
5. Commit mit klarem Scope

## Verboten

- Arbeit als fertig melden, obwohl das Bead nicht aktualisiert ist
- bekannte Folgearbeit nur im Chat hinterlassen
- stillschweigend Scope ausweiten
