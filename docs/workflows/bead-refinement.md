# Bead Refinement Workflow

## Ziel

Beads muessen so klein und klar sein, dass ein Agent sie ohne Rueckfrage bearbeiten kann.

## Checkliste

1. Problem klar?
2. Ziel beobachtbar?
3. Nicht im Scope explizit?
4. Akzeptanz testbar?
5. Abhaengigkeiten gesetzt?
6. Folgefragen ausgelagert?

## Regel

Wenn ein Bead mehrere unabhaengige Ergebnisse erzeugt, ist es meist zu gross.
